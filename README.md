 Script: Create Debian LXC and Install OpenClaw on Proxmox
# Description: Automates the creation of a Debian LXC container on Proxmox.
# Auto-detects storage and installs OpenClaw using the CORRECT URL.


How to run:
1. Download or bash -c "$(curl -fsSL [https://raw.githubusercontent.com/community-scripts/ProxmoxVE/main/ct/projectsend.sh)](https://raw.githubusercontent.com/zorrong/Proxmox-script/refs/heads/main/install_openclaw.sh)"

3. chmod +x install_openclaw.sh
4. Save and run: ./install_openclaw.sh
